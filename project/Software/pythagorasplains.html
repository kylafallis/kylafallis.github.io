Pythagorus Plains

An interactive educational game built with vanilla JavaScript.

Project Narrative

Pythagorus Plains is a web-based educational game I developed to make learning geometric principles more accessible and fun. The core of the game involves a series of interactive, puzzle-based challenges where users apply mathematical concepts to progress.

Lessons Learned

Building this project was a fantastic exercise in DOM manipulation and state management with vanilla JavaScript. I focused on creating a real-time feedback loop for the user, ensuring the interface was responsive and intuitive for both desktop and mobile use. It taught me a great deal about structuring game logic and designing for a positive user experience.

Tech Stack

  • HTML5
  • CSS3
  • JavaScript (Vanilla)
