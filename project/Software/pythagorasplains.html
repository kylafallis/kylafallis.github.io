Project Overview

About this Project

Pythagorus Plains is a web-based educational game designed to teach geometric principles in an interactive and engaging way. The project was developed to make learning math more accessible and fun for students.

Key Features

  • Interactive, puzzle-based challenges
  • Real-time feedback for users
  • Responsive design for both desktop and mobile use

Technologies Used

This project was built from scratch using HTML, CSS, and vanilla JavaScript.
